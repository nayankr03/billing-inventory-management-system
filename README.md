# 🧾 Billing Inventory Management System (BIMS)

A modern **Billing Inventory Management System (BIMS)** built with **Core PHP, MySQL, Bootstrap 5, JavaScript, HTML, and CSS**. It helps businesses manage products, customers, sales, inventory, invoices, users, and reports through a simple and responsive web interface.

# 💻 Installation

### Clone Repository

```bash
git clone https://github.com/nayankr03/billing-inventory-management-system.git
```

### Move Project

Copy the project into:

```
C:\xampp\htdocs\
```

### Import Database

- Open phpMyAdmin
- Create a database
- Import the SQL file

### Configure Database

Update:

```
includes/config.php
```

with your MySQL credentials.

### Run Project

```
http://localhost/PROJECT/
```
